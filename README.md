# noctalia-plugins

Noctalia Shell 插件集合。

## 插件列表

### [developer-tools](./developer-tools)

开发者工具集合，提供常用转换和格式化功能：
- 时间戳转换（Unix timestamp ↔ 日期时间）
- JSON 格式化与验证

### [query-tracker](./query-tracker)

Shell 命令执行结果监控器：
- 配置需要监控的命令
- 定期执行并捕获输出
- 在状态栏显示执行状态（成功/失败计数）

### [modifier-keys](./modifier-keys)

键盘修饰键状态与触摸板手势显示：
- 实时显示 Super、Alt、Ctrl、Shift 键状态
- 显示最近按下的普通按键
- 触摸板手势识别（滚动、滑动、点击）
